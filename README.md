# 🎬 Cinema Data Analysis – Lombardy

## 📊 Project Overview
This project analyzes cinema data from the **Lombardy region** using the file:

📁 `Exercise 1.4 - Cinema.xlsx`

The dataset contains:
- 🎟️ Number of tickets sold  
- 💰 Total receipts (in USD)  
- 📍 Data categorized by province  

---

## 🎯 Objectives

✔️ Perform sorting and calculations in Excel  
✔️ Analyze ticket sales and revenue  
✔️ Derive insights using statistical measures  

---

## 📝 Tasks Breakdown

### 🔽 1. Sort Data
- Sort the table based on **Takings (Receipts)**  
- Order: **Descending (Highest → Lowest)**  

---

### ➕ 2. Total Calculation (Row 17)

📌 Calculate:

- **Total Tickets Sold**
- **Total Takings (USD)**

💡 Formula Example:
```excel
=SUM(B2:B16)   // Tickets
=SUM(C2:C16)   // Takings
```

## 📉 3. Statistical Analysis

Calculate the following for both **Tickets** and **Takings**:

- 🔻 Minimum Value  
- 🔺 Maximum Value  
- 📊 Average Value  

💡 **Formula Example:**
```excel
=MIN(B2:B16)
=MAX(B2:B16)
=AVERAGE(B2:B16)
```

## 🔢 4. Count Provinces

📌 Count total number of provinces in Lombardy  

💡 **Formula:**
```excel
=COUNTA(A2:A16)
```

## 💵 5. Average Ticket Price

📌 **Formula:**

```excel
= Total Takings / Total Tickets
```

## 🛠️ Tools Used

- 📗 Microsoft Excel  
- 📊 Functions: `SUM`, `MIN`, `MAX`, `AVERAGE`, `COUNTA`
